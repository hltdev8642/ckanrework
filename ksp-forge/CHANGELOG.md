# Changelog

All notable changes to KSP Forge will be documented in this file.

## [0.5.8] - 2026-04-22
### Changed
- Incremented release version to `0.5.8`
- Fixed CKAN tags filter so selecting multiple tags now matches mods containing all selected tags
- Fixed the reset button so CKAN tag selections, filter state, and CurseForge filters are fully cleared
- Improved filter state persistence and UI behavior in Discover

### Documentation
- Updated `README.md` to display the current release version

## [0.5.7] - 2026-04-21
### Changed
- Updated Windows build version to `0.5.7`
- Added CurseForge REST API key support and settings persistence
- Added CurseForge sync caching to reduce repeated API calls
- Expanded CurseForge search paging for more than 50 results

### Documentation
- Added release notes pointer to `README.md`
- Added `CHANGELOG.md` for future releases
